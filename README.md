## Random News Generator API

A FastAPI-based backend application that generates category-based fake news headlines using dynamic templates.

### Features
- Category & sentiment-based news generation
- RESTful APIs
- SQLite database for history
- Clean project architecture

### Tech Stack
- Python
- FastAPI
- SQLite
- Pydantic

### Run Project
pip install -r requirements.txt  
uvicorn app.main:app --reload
